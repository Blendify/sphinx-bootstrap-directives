Tables

Basic Tables
============

Basic tables can be styled by adding the `:class: table` to rst table directives.

.. list-table:: Frozen Delights!
   :widths: 15 10 30
   :header-rows: 1
   :class: table

   * - Treat
     - Quantity
     - Description
   * - Albatross
     - 2.99
     - On a stick!
   * - Crunchy Frog
     - 1.49
     - If we took the bones out, it wouldn't be
       crunchy, now would it?
   * - Gannet Ripple
     - 1.99
     - On a stick!
