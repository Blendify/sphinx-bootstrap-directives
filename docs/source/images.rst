******
Images
******

.. todo add examples

Images can be changed to work with bootstrap by add the `:class:` atrribute
to `.. image::` or `.. figure::` directives. Supported options are:

- `img-rounded`
- `img-circle`
- `img-thumbnail`
- `img-responsive`

  .. note:: This will not work if you set the height or the width of the image.
